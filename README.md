# High-Performance URL Shortener

## Overview
A scalable URL shortening service built with Go, PostgreSQL, and Redis.

## Features
- URL shortening
- Click tracking
- Caching
- Rate limiting

## Getting Started
\`\`\`bash
# Clone the repository
git clone https://github.com/shawgichan/url-shortener.git

# Start services
docker-compose up --build
\`\`\`

## Development
- Golang 1.21+
- Docker
- PostgreSQL
- Redis
